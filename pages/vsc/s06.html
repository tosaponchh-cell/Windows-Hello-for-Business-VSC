06 — VSC SETUP

Create Virtual Smart Card (CLIENT01)

ทำบน CLIENT01 — ต้อง domain-joined, มี TPM พร้อมใช้งาน, run CMD as Administrator

VSC Creation Flow

1
ตรวจสอบ TPM Status

Windows + R → tpm.msc → Status ต้องเป็น: "TPM is ready for use"
ถ้าไม่ ready: shut down VM → add Trusted Platform Module ใน VM hardware → restart

2
Open CMD as Administrator — Critical

Windows key → search Command Prompt → Right-click → Run as administrator
⚠ รันโดยไม่ elevation = Error: Access is denied (0x80070005)

3
Run Create Command 
tpmvscmgr.exe create /name "LabVSC" /pin default /adminkey random /generate
// /pin default = PIN ถูกตั้งเป็น 12345678
// /pin prompt  = ตั้ง PIN เอง (minimum 8 characters)
// Expected: Command completed successfully
// บันทึก Instance ID: ROOT\SMARTCARDREADER\0000
4
Verify ใน Device Manager

Windows + X → Device Manager → ขยาย Smart card readers
ต้องเห็น: Microsoft Virtual Smart Card 0
ถ้าไม่มีใน list: ห้าม proceed to enrollment — Re-create VSC ก่อน

Command Reference

// Create VSC with default PIN 12345678
tpmvscmgr.exe create /name "LabVSC" /pin default /adminkey random /generate

// Create VSC with custom PIN (min 8 chars)
tpmvscmgr.exe create /name "LabVSC" /pin prompt /adminkey random /generate

// Delete VSC by instance ID
tpmvscmgr.exe destroy /instance ROOT\SMARTCARDREADER\0000
⚠ .exe ไม่ใช่ .msc

tpmvscmgr.exe ไม่ใช่ tpmvscmgr.msc — extension .msc คือ Management Console snap-in ไม่ใช่ command ที่ run ได้ ใช้ .exe เสมอ
