Kerberos PKINIT — Login Sequence

#ActorAction
1CLIENT01User enters PIN → TPM unlocks private key on VSC
2CLIENT01Creates cryptographic signature + attaches user certificate → builds AS_REQ
3CLIENT01 → DC01Sends AS_REQ to DC01 on Kerberos port 88
4DC01 (KDC)Traces cert chain: LabUser01 cert ← Issuing CA ← Root CA
5DC01 (KDC)Downloads CRL from LDAP — checks cert is NOT revoked ← MOST COMMON FAILURE
6DC01 (KDC)Looks up LabUser01@lab.local in Active Directory
7DC01 (KDC)All checks pass → issues Ticket-Granting Ticket (TGT)
8DC01 → CLIENT01Sends TGT back to CLIENT01
9CLIENT01Uses TGT to request session ticket → loads LabUser01 desktop
