05 — PKI BUILD

Certificate Template

ทำบน ISSUINGCA — template นี้กำหนด rules สำหรับ Smart Card certificates ที่ออกให้ user

⚠ CRITICAL — Security Tab

Security tab คือ tab ที่ถูก miss บ่อยที่สุด ถ้าไม่ตั้ง Read + Enroll permissions สำหรับ Authenticated Users → template จะ invisible กับ user ทุกคนตอน enrollment

All Five Template Tabs

TabSettingValue ที่ต้องตั้ง
CompatibilityCA CompatWindows Server 2012 | Recipient: Windows 10 / Server 2016
GeneralKey SettingsName: Lab SmartcardLogon01 | Validity: 1 Year | Publish cert in AD: ENABLED
Request HandlingPurposeSignature and Smartcard Logon | Prompt user during enrollment: YES
CryptographyProviderKey Storage Provider — Microsoft Smart Card Key Storage Provider ONLY | RSA 2048 | SHA256
SecurityPermissionsAuthenticated Users: READ ✔ ENROLL ✔

Step-by-Step

1
Open certtmpl.msc และ Duplicate Template

Windows + R → certtmpl.msc → หา Smartcard Logon → Right-click → Duplicate Template

2
Configure ทุก Tab ตามตารางด้านบน

ทำตามลำดับทุก tab โดยเฉพาะ Security tab ต้องทำเสมอ

3
Publish Template ไปที่ Issuing CA

certsrv.msc → ขยาย CA name → Right-click Certificate Templates
→ New → Certificate Template to Issue → เลือก Lab SmartcardLogon01 → OK
ตรวจว่า template name ปรากฏใน Certificate Templates folder

💡 Template ไม่ปรากฏใน Issue list

รอ 2 นาทีสำหรับ AD replication จากนั้น stop และ start CA service แล้ว refresh
