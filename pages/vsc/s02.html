02 — PKI BUILD

Build Root CA (ROOTCA)

ทำบน ROOTCA เท่านั้น — เครื่องนี้ต้องไม่ join domain และต้อง shut down เมื่อไม่ใช้งาน

📌 INFO

ทำบน ROOTCA เท่านั้น เครื่องนี้ต้องไม่ join domain — Keep powered off when not in use

Root CA Setup Flow

1
Install AD CS Role

Server Manager → Manage → Add Roles and Features → Active Directory Certificate Services
Role Services: เลือก Certification Authority only

2
Configure: Standalone Root CA

Click yellow flag → Configure AD CS
Setup Type: Standalone CA (Enterprise CA greyed out — ROOTCA ไม่ join domain)
CA Type: Root CA | Private Key: Create new | RSA 2048, SHA-256
CA Name: LAB-RootCA | Validity: 10–20 years

3
Publish CRL ก่อน Export (สำคัญมาก)

Open certsrv.msc → Right-click Revoked Certificates → All Tasks → Publish → New CRL → OK
ถ้าไม่ทำขั้นตอนนี้ CRL อาจหมดอายุก่อนที่จะ export

4
Export ทั้งสองไฟล์และ Copy ออก

ไฟล์อยู่ที่: C:\Windows\System32\CertSrv\CertEnroll\
Copy ทั้งสองไฟล์: LAB-RootCA.crt + LAB-RootCA.crl
ปลายทาง: ISSUINGCA และ DC01 (ทั้งสองเครื่องต้องการไฟล์เหล่านี้)

5
Shut Down ROOTCA ทันที

Shut down ROOTCA ทันทีหลัง copy ไฟล์เสร็จ บันทึก CRL Next Update date และตั้ง calendar reminder
