07 — VSC SETUP

Certificate Enrollment

ต้อง login เป็น LabUser01 (domain user) ไม่ใช่ Administrator

⚠ CRITICAL — Login เป็น User ถูกต้อง

Enrollment ต้องทำขณะ login เป็น LabUser01 เท่านั้น ไม่ใช่ Administrator cert ต้องอยู่ใน LabUser01's Personal store มิฉะนั้น Smart Card login จะล้มเหลว

Enrollment Flow

1
Log In เป็น LabUser01

Log off หรือ switch user → click Other user
Username: lab\LabUser01 | Password: [AD password]

2
Open certmgr.msc (User Store)

Windows + R → certmgr.msc → Enter
นี่คือ Current User certificate store (ไม่ใช่ local machine store)

3
Request New Certificate

ขยาย Personal → Right-click Certificates → All Tasks → Request New Certificate...
Click Next → Select Active Directory Enrollment Policy → Next
Tick Lab SmartcardLogon01 → Click Enroll

4
Enter PIN เมื่อถูกถาม

Windows Security dialog ปรากฏขึ้นขอ smart card PIN
Enter: 12345678 (ถ้าใช้ /pin default) → Click OK

5
Verify Success

Status: Succeeded → click Finish
ใน certmgr.msc → Personal → Certificates: cert ของ LabUser01 issued by LAB-IssuingCA ต้องปรากฏ
Double-click cert → Details → Enhanced Key Usage: Smart Card Logon + Client Authentication
Subject Alternative Name: LabUser01@lab.local
