WHfB vs FIDO2 Security Keys

เลือกให้ถูก use case — อย่าซื้อ hardware token ให้ทุกคนโดยไม่จำเป็น

มิติWHfB (TPM-based)FIDO2 Security Key
ค่าใช้จ่ายฟรี — ใช้ TPM ที่มีอยู่แล้ว$25–$70 ต่อ key
PlatformWindows + TPM 2.0 เท่านั้นWindows, macOS, Linux, iOS, Android
Shared PC / Kioskไม่เหมาะ (ผูกกับ 1 เครื่อง)เหมาะมาก (พก key ไปทุกเครื่อง)
ย้ายเครื่องบ่อยไม่สะดวกพก key ก้อนเดียวใช้ได้ทุกที่
Lost / Stolenkey อยู่ใน TPM ขโมยไม่ได้ทำหาย = revoke + issue ใหม่
BiometricWindows Hello built-inเฉพาะ key ที่มี bio sensor
On-prem KerberosKey Trust / Cloud Kerberosจำกัด
Admin overheadต่ำ (Intune/GPO)สูงกว่า (inventory, lost key)
